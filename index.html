Git commands

$ git add . - commits changes to the local staging area

$ git status - tells you the status of filkres in the current branch

$ git checkout [develop] [feature/XX-XXX]

Git commands to commit and push to GitHub

$ git add .

$ git commit -m "type message in quotes"
If prompted to add a comment you press i to type your comment then esc to leave. :wq also takes you back to the comand line.

$ git checkout develop

$ git pull

$ git checkout feature.CD-XXXX

$ git merge develop

$ git push

The feature should now be visible in GitHub and can be merged into develop. WARNING - check that you are merging into develop and not master

Commit commands

$ git commit -m "Message explaining what the commit is" - commits files to the local staging area

$ git commit -a -m "Message about committing an already committed file" - if a file has already been committed, then this command can be used to commit them again. -a adds the files at the same time, so does not require you to 'git add .' them first

$ git reset --soft HEAD^ - will undo a commit.

$ git add forgotten-file.html - to add a file that you forgot.

$ git commit --amend -m "New messga related to the commit" - lets you update the commit message to include the forgotten file.
